## Logarithmic

| fx         | f'x                   |
| ---------- | --------------------- |
| $e^x$      | $e^x$                 |
| $a^x$      | $a^x\;ln\;a$          |
| $ln\;x$    | $\frac{1}{x}$         |
| $log_a\;x$ | $\frac{1} {x\;ln\;a}$ |

## Trigonometric
| Easy       | Trick              |
| ---------- | ------------------ |
| sin = cos  | sec -> sec <- tan  |
| cos = -sin | csc -> -csc <- cot |

| fx       | f'x               |
| -------- | ----------------- |
| $sin\;x$ | $cos\;x$          |
| $cos\;x$ | $-sin\;x$         |
| $tan\;x$ | $sec^2\;x$        |
| $cot\;x$ | $-csc^2\;x$       |
| $sec\;x$ | $sec\;x\;tan\;x$  |
| $csc\;x$ | $-csc\;x\;cot\;x$ |

## Hyperbolic
| Hyperbolic                        | Trick                    |

| $cosh(x) = \frac{e^x+e^{-x}} {2}$ | c**o**sh == p**o**sitive |
| $sin(x) = \frac{e^x-e^{-x}} {2}$  | si**n**h == **n**egative |

## Partial Derivatives
* Treat other variables as constants
* E.g. $f'x(x^2y^4) = 2xy^4$
## Anti-derivative
* Undos power rule
$$
\begin{aligned}
Form:\\
bx^c &= \frac{bx^{c+1}}{c+1}\\
Example:\\
6x^2 &= 2x^3
\end{aligned}
$$
## Definite Integral
* A function's area under a curve
$$
\int^{b}_{a}\;f(x)\;dx = \int\,f(b) - \int\,f(a)\
$$
